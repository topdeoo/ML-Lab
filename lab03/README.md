## 实验数据

使用了《统计学习方法》中决策树部分的数据，是一个银行评估人是否可以通过贷款申请（也就是类别）。

数据集含义解释：

ID：数据的编号

年龄：0表示青年，1表示中年，2表示老年

有工作：1表示有，0表示无

有自己的房子：1表示有，0表示无

信贷情况：0表示一般，1表示好，2表示非常好

贷款申请：1表示是，0表示否
